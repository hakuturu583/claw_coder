# Repo Orientation

Use this skill when the task targets a repository checkout.

- Inspect `git status` before making changes.
- Read the repository-local `AGENTS.md` before editing code.
- Identify the primary entrypoints, config files, and tests relevant to the task.
- Prefer small, focused changes and avoid destructive git operations.
